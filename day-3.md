### Day 3

# Today I learned some basic concepts in Python like variables, data types and operators.

# Variables
A variable is used to store data which can be used later in a program.
Example:
name = "Impana"
age = 20

# Data Types

# Integer (int)
Used to store whole numbers
Example: x = 10

# Floating point (float)
Used to store decimal numbers
Example: price = 99.5

# String (str)
Used to store text or characters
Example: name = "Python"

# Boolean (bool)
Used to store True or False values
Example: is_valid = True

# None type
Represents no value
Example: value = None

## Rules for naming variables

* A variable name can contain alphabets, digits, and underscore.
* A variable name can only start with an alphabet and underscore.
* a variable name can't start with a digit.
* No spaces is allowed to be used inside a variable name.

# Examples:
valid_name = "Yes"
_age = 20
user1 = "Hello"

# Operators in Python

* Arithmetic operators
Used for calculations
+, -, *, /

* Assignment operators
Used to assign values
=, +=, -=

* Comparison operators
Used to compare values
==, !=, >, <, >=, <=

* Logical operators
Used to combine conditions and give True or False

* AND operator
Gives True only if both conditions are True

Truth table:
True and True = True
True and False = False
False and True = False
False and False = False

Example:
a = 10
print(a > 5 and a < 20)

* OR operator
Gives True if at least one condition is True

Truth table:
True or True = True
True or False = True
False or True = True
False or False = False

Example:
a = 10
print(a > 5 or a < 5)

* NOT operator
Reverses the result

Truth table:
not True = False
not False = True

Example:
a = 10
print(not(a > 5))
